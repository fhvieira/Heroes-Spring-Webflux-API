# Heroes-Spring Webflux-API
Heroes API using Spring WebFlux - DIO Class

Run project:
```bash
mvn spring-boot:run 
```

Run tests:
```bash
mvn clean test
```

Project start endpoint:
```bash
http://localhost:8080/heroes
```

Project documentation:
```bash
http://localhost:8080/swagger-ui-reactive-api.html
```

## Technologies:
```bash
Java 11
Maven
Spring WebFLux
DynamoDB
JUnit
Reactor
```
